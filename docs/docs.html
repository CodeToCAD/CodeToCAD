Materials affect the appearance and simulation properties of the parts.
assignToPart() setColor() addImageTexture()
Animation
Animation related functionality.
default() setFrameStart() setFrameEnd() setFrameCurrent() createKeyFrameLocation() createKeyFrameRotation()
Light
Manipulate a light object.
setColor() createSun() createSpot() createPoint() createArea() translateXYZ() rotateXYZ() isExists() rename() delete() getNativeInstance() getLocationWorld() getLocationLocal() select()
Camera

createPerspective() createOrthogonal() setFocalLength() translateXYZ() rotateXYZ() isExists() rename() delete() getNativeInstance() getLocationWorld() getLocationLocal() select()
Render
Render the scene and export images or videos.
renderImage() renderVideoMp4() renderVideoPng() setFrameRate() setResolution() setRenderQuality() setRenderEngine()
Scene
Scene, camera, lighting, rendering, animation, simulation and GUI related functionality.
default() create() delete() getSelectedEntity() export() setDefaultUnit() createGroup() deleteGroup() removeFromGroup() assignToGroup() setVisible() setBackgroundImage()

Tools for collecting data about the entities and scene.
measureDistance() measureAngle() getWorldPose() getBoundingBox() getDimensions() log()
